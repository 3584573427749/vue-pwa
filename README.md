# Vue PWA Template

Detta är ett GitHub‑template‑repo för PWA-appar i plattformen.

## Funktioner
- Vue 3 + Vite
- Installationsbar PWA (manifest + service worker)
- Offline-stöd & caching
- DTO-typer via OpenAPI (generated.d.ts)
- Docker-image med Nginx
- ESLint
- VERSION-fil (semver)

## Kör lokalt
```bash
npm install
npm run dev
```

## Bygg
```bash
npm run build
```
